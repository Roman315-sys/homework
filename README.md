# Разработка фичи для личного кабинета в банковском приложении

## Описание:
 На данный момент в проекте реализован ряд полезных функций!
1) Сокрытие номера карты или счета в зависимости от типа.
2) Фильтр списков по значению (state) сортировка списков по указанному полю с датой.
3) В проекте реализованы генераторы (примеры их использования будут ниже). Благодаря генераторам пользователь сможет - сортировать историю по валюте, генерировать номер карты, видеть историю переводов!
## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Roman315-sys/homework.git
```
2. Установите зависимости:
```
poetry install
```

## Тестирование:
1. Для просмотра результатов тестов используйте команду:
```
pytest --cov
```

## Генераторы:
В проекте реализовано 3 генератора!
1) filter_by_currency позволяет фильтровать данные по валюте. Пользавателю достаточно передать словарь и необходимую для сартировки валюту.
2) transaction_descriptions возвращает пользавателю историю переводов.
3) card_number_generator генерирует номер карты в заданном диопазоне. Пользавотелю для получения желаемого номера надо продублировать этот номр в качестве параметра функции.
